Финансовый тамагочи с вечными пассивными фермами, бухгалтерией и AI-комментариями.  
**Бот для личного пользования (доступен только для одного ID).**

---

## ✨ Возможности

### 🏦 Экономическая система
- **4 валюты**: Рубли (RUB), Доллары (USD), Евро (EUR), Биткоин (BTC)
- **Вечные фермы**: Включил и забыл — работают 24/7
- **Пассивный доход**: Начисляется автоматически каждые 6 часов
- **Апгрейд ферм**: Улучшай доходность до бесконечности
- **Курсы валют**: Меняются каждый день (имитация рынка)
- **Максимум 2 активные фермы одновременно**

### 📚 Бухгалтерия
- **Ежедневные задачи**: Решай уравнения разной сложности
- **4 уровня сложности**: Лёгкая → Нормальная → Сложная → Профессионал
- **Streak система**: Чем дольше не пропускаешь, тем выше множитель
- **Множители дохода**: x1.25 → x1.5 → x2.0 → x3.0 и выше
- **Бесконечное прохождение**: Засчитываются только первые 2 раза в день

### 🎯 Система сложности (перевернутая)
- **На высокой сложности — ниже бонус дохода, но ниже комиссия на обмен**
- **Easy**: бонус x0.8, комиссия 3%
- **Normal**: бонус x1.0, комиссия 2%
- **Hard**: бонус x0.6, комиссия 1.5%
- **Pro**: бонус x0.5, комиссия 1%
- **Ошибки на высокой сложности сбрасывают Streak**

### 🤖 Искусственный интеллект (1000+ фраз)
- **4 стиля общения**: Дружелюбный, Деловой, Саркастичный, Мотивирующий
- **Частота ответов**: Настрой от 1% до 100%
- **Живые комментарии**: Реагирует на все действия
- **Случайные советы**: Сотни комбинаций фраз
- **Настраиваемое имя бота**

### 📊 Статистика и достижения
- **Прогресс-бары**: Визуализация баланса
- **Система достижений**: 20+ различных ачивок
- **История транзакций**: Все операции сохраняются
- **Детальная аналитика**: По каждой ферме и валюте

### 🔒 Безопасность
- **Ограничение доступа**: Только для одного Telegram ID (6595788533)
- **Глобальная обработка ошибок**: Бот не падает при исключениях
